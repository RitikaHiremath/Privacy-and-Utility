# Privacy-and-Utility

## Utility Score

The Utility Score measures the extent to which a model's completion meets a task relative to a perfect response. It is typically assessed utilizing ROUGE (Recall-Oriented Understudy for Gisting Evaluation).

## Privacy Score

The Privacy Score measures how much a prompt or response conceals or alters sensitive original information.Levenshtein Distance measures the least number of single-character modifications (insertions, deletions, or replacements) needed to change one string into a different one. 1 indicates increased utility, while 1 signifies enhanced privacy.
